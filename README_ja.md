# クーリングクレジット定義

[English](README.md) | [日本語](README_ja.md) | [العربية](README_ar.md)

## 国際標準案としての定義・分類フレームワーク

**クーリングクレジット**とは、既存の熱負荷を直接低減すること、または地球システムに備わる自然冷却機能を回復させることに対して発行される、測定・検証可能なクレジット単位である。

この名称は、気温を直接下げる取り組み、または水の相転移、潜熱輸送、蒸散、土壌水分、海洋・大気熱交換などの自然冷却機能を補完・回復・再起動する取り組みに限定して用いるべきである。

## 公式定義

クーリングクレジットは、周辺気温、地表面温度、水温、既存熱負荷、水の相転移、潜熱輸送、土壌水分、蒸散、海洋・大気間の熱交換などについて、測定・報告・検証できる冷却成果がある場合にのみ成立する。

クーリングクレジットが問うのは、どれだけCO₂を削減・除去・相殺したかではない。どれだけ熱負荷、温度ストレス、水循環の乱れ、土壌水分の喪失、自然冷却機能の低下を物理的に減らしたかである。

## 名称保護条項

**Cooling Credit / クーリングクレジット** は、直接冷却または自然冷却機能の回復に限定される名称である。

Sunshade Credit / サンシェードクレジット、Reflect Credit / リフレクトクレジット、Carbon Credit / カーボンクレジットは別分類であり、直接冷却成果が別途検証されない限り、Cooling Credit / クーリングクレジットではない。

この区別は、概念の希釈、グリーンウォッシュ、会計上の排出削減と物理的冷却成果の誤った同一視を防ぐために必要である。

## 分類ルール

- Cooling Credit: 直接的な気温低下、既存熱負荷の低減、または自然冷却機能の回復。
- Carbon Credit: CO₂の削減・除去・相殺。直接冷却成果が別途検証されない限り、Cooling Creditではない。
- Sunshade Credit: 太陽入力の低減を主目的とする分類。Cooling Creditではない。
- Reflect Credit: 反射率やアルベドの上昇を主目的とする分類。直接冷却成果が別途検証されない限り、Cooling Creditではない。
- Adaptation Credit / Resilience Credit: 脆弱性低減や防災強化を目的とする分類。物理的冷却が検証されない限り、Cooling Creditではない。

## 要約

クーリングクレジットは、カーボンクレジットではない。  
クーリングクレジットは、サンシェードクレジットではない。  
クーリングクレジットは、リフレクトクレジットではない。

クーリングクレジットとは、物理的冷却、または自然冷却機能の回復を検証した単位である。

気温を直接下げる、既存熱負荷を下げる、または地球本来の自然冷却過程を回復する取り組みだけを、クーリングクレジットと呼ぶべきである。
