# 冷却名称境界定義

## 冷却・遮光・反射・熱入力抑制は分けて定義する必要がある

本書は、**Cooling Credit（クーリングクレジット）** という名称を守るための厳格な境界定義である。

**冷却**という言葉は、現在すでに地球システム内に存在する熱負荷を直接下げる行為、自然冷却機能を回復する行為、または熱を移動・放出・循環・散逸させる自然現象を再起動・補完する行為に限定するべきである。

単に太陽光を遮る、影を作る、反射する、将来入ってくる日射を一時的に減らすだけの行為を、冷却と呼ぶべきではない。

---

## 1. 中核定義

クーリングクレジットにおいて **冷却** と呼べるのは、次の行為だけである。

```text
今ある地球の熱負荷を下げる行為、
または、自然現象・水の相転移・水循環を実際に起こす、
あるいは補完して自然冷却機能を再起動させる取り組み。
```

これには、次のような成果を生む、または回復する行為が含まれる。

- 既存熱負荷の低減
- 気温、地表温度、土壌温度、水温、海面水温の低下
- 蒸散・蒸発散の回復
- 気化熱による潜熱輸送
- 土壌水分の回復
- 保水・水循環の回復
- 自然過程の回復による雲・雨・水循環の再起動
- 海洋混合、酸素供給、熱交換の回復
- 人工排熱の低減
- 森林、土壌、湿地、都市、海洋における自然冷却機能の回復

---

## 2. 冷却と呼ぶべきではないもの

次のものは、原則として **クーリングクレジット** ではない。

- 太陽光遮蔽
- 宇宙日傘
- 成層圏エアロゾル散布
- 人工的な日射遮断
- 鏡・反射板による反射システム
- アルベド上昇のみの施策
- 白色化・反射率向上のみの施策
- 実測された熱負荷低減を伴わない日陰づくり
- 入ってくる日射を減らすだけで、既存熱負荷の低下や自然冷却機能回復を伴わない事業

これらは、必要であれば別分類で議論するべきである。

例：

- サンシェードクレジット
- シェードクレジット
- リフレクトクレジット
- アルベドクレジット
- 熱入力抑制クレジット
- 太陽放射管理クレジット

これらを自動的にクーリングクレジットと呼ぶべきではない。

---

## 3. なぜ区別が必要なのか

太陽光を遮ることと、地球を冷やすことは同じではない。

日傘は、将来入ってくる日射の一部を減らすかもしれない。  
反射板は、太陽光の一部を反射するかもしれない。  
エアロゾル層は、放射収支を変えるかもしれない。

しかし、それらは自動的に次を実現するわけではない。

- 海洋に蓄積した熱を取り除く
- 都市に蓄積した熱を減らす
- 土壌水分を回復する
- 蒸散・蒸発散を再起動する
- 森林の冷却機能を回復する
- 水循環を回復する
- 海洋の酸素供給を回復する
- 人工排熱を減らす
- 地域住民が実際に涼しさを感じる熱ストレス低減を生む
- 地域で測定可能な冷却成果を生む

したがって、これらをクーリングクレジットと同一視してはならない。クーリングクレジットと呼ぶには、定義、境界ルール、MRV要件を独立して満たす必要がある。

---

## 4. 水の相転移原則

地球の自然冷却は、水と相転移に深く関係している。

重要な自然冷却過程には、次がある。

- 蒸発
- 蒸散
- 蒸発散
- 凝結
- 雲形成
- 降雨
- 土壌保水
- 地下水涵養
- 海洋・大気熱交換
- 潜熱輸送

クーリングクレジットの適格性は、これらの過程を回復または補完する行為を優先するべきである。

例：

- 土壌水分を回復し、蒸発・植物蒸散を再開させる
- 森林や混交植生を回復し、蒸発散を戻す
- 湿地、河川、都市水路を回復する
- 雨水、再生水、循環水を用いて気化冷却を行う
- 建築物、都市インフラ、産業からの排熱を減らす
- 厳格な生態系監視の下で海洋循環と酸素供給を支援する

---

## 5. 名称ルール

名称ルールは単純である。

```text
既存熱負荷を下げる、または自然冷却過程を回復するなら、
クーリングクレジットとして評価できる可能性がある。

太陽光を遮るだけ、日陰を作るだけ、放射を反射するだけなら、
遮光、サンシェード、反射、アルベド、熱入力抑制と呼ぶべきである。
```

**Cooling Credit** という言葉を、温暖化を遅らせると主張するあらゆる事業のマーケティング用語にしてはならない。

---

## 6. 市場健全性ルール

ある事業が「クーリング」「気候冷却」「地球冷却」という言葉を使っていても、それだけでクーリングクレジットとして販売してはならない。

適格とするには、次を示す必要がある。

1. 検証済みの冷却、または自然冷却機能の回復
2. 明確な基準線
3. 測定可能な指標
4. MRV
5. リスク開示
6. 持続期間と有効期間
7. カーボンオフセット、太陽光遮蔽、反射率向上のみの主張との区別

これらがなければ、クーリングクレジットとして発行すべきではない。

---

## 7. 要約

クーリングクレジットは、次の行為に限定するべきである。

```text
今ある地球システムの熱負荷を下げる行為、
または自然冷却現象、特に水循環と水の相転移を再起動・回復する行為。
```

太陽光遮蔽、宇宙日傘、鏡・反射板、アルベド向上のみの施策、成層圏エアロゾル的手法は、入射日射に影響を与える可能性はあるが、それ自体ではクーリングクレジットではない。

それらは、MRVによって物理的な冷却成果と自然冷却機能回復を示さない限り、**Cooling / 冷却** ではなく、**Sunshade / 遮光**、**Shade / 日陰**、**Reflect / 反射**、**Albedo / アルベド**、**Heat-Input Suppression / 熱入力抑制** と呼ぶべきである。
