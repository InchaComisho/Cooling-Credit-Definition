# クーリングクレジット — 国際標準草案

> **注記：** この文書は提案された標準草案であり、公式に採択された標準ではありません。マスター / inchacomusho / InchaComisho が概念的・構造的フレームワークとして提案するものです。

---

## 1. 目的

クーリングクレジットを、物理的な冷却成果または自然冷却機能の回復を測定・報告・検証する独自のクレジットカテゴリとして定義すること。カーボンクレジット、太陽地球工学クレジット、適応クレジットとは区別されます。

---

## 2. 定義

クーリングクレジットとは、以下の成果の一つ以上を表す、測定・検証・帰属可能な単位です。

1. 周辺気温の直接低下
2. 地表温度の直接低下
3. 水温または海面温度の直接低下
4. 定義された地域またはシステムにおける既存熱負荷の低減
5. 水の相変化（蒸発、凝縮、雲形成、降水支援、凍結、融解、潜熱輸送）の回復・支援
6. 土壌水分、植生蒸散、または水文緩衝機能の回復
7. 海洋・大気熱交換と循環の回復・支援
8. 測定可能な冷却または冷却機能回復による気候関連リスク圧の低減

---

## 3. 適用範囲

この標準は、クーリングクレジットの発行を主張するプロジェクト、活動、介入に適用されます。適格基準、境界ルール、MRV要件、除外カテゴリを定義します。

---

## 4. 他クレジットとの区別

| クレジット種別 | 主な仕組み | クーリングクレジットか？ |
|---|---|---|
| クーリングクレジット | 直接的な温度低下または自然冷却機能の回復 | はい |
| カーボンクレジット | CO₂削減・除去・相殺 | いいえ（個別に検証された直接冷却成果がある場合を除く） |
| 太陽遮蔽クレジット | 太陽放射の遮断・低減 | いいえ |
| 反射クレジット | 地表反射率・アルベドの増加 | いいえ（個別に検証された直接冷却成果がある場合を除く） |
| 適応クレジット | 脆弱性・暴露の低減 | いいえ（物理的冷却が検証された場合を除く） |
| 強靭性クレジット | 防災準備の改善 | いいえ（物理的冷却が検証された場合を除く） |
